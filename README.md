# 🎂 Happy Birthday My Love ❤️

A beautiful, interactive birthday surprise website built with **Next.js 15**, **React 19**, and **Tailwind CSS**.

## ✨ Features
- **Hero Section**: A grand entrance with a "TAP TO ENTER" portal.
- **Background Music**: Slow romantic piano (YouTube) with a custom visualizer.
- **Interactive Quizzes**: Fun questions about our favorite things to unlock memories.
- **3D Effects**: Interactive kiss effects (💋) and a 3D gift box (🎁) opening animation.
- **Photo Gallery**: A beautiful grid of our best moments with lightbox view.
- **Journey Carousel**: A moving timeline of our story together.
- **Faith & Dreams**: Dedicated sections for her devotion to Shri Krishna and her dreams.
- **Secret Letter**: A locked message that opens with a special magic word.
- **Responsive Design**: Works perfectly on mobile and desktop.

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🛠️ Built With
- **Next.js 15** (App Router)
- **React 19**
- **Tailwind CSS**
- **Framer Motion** & **Canvas Confetti**
- **Lucide React** (Icons)

## 📸 Personalization
To use your own photos:
1. Place your images in the `public/Rose/` directory.
2. Update the image paths in `app/page.tsx`.
